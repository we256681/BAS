License management through web interface. User management
?

Premium users can manage customers licenses through web interface. Licenses for BAS scripts may be limited in time. Premium developer can stop user license manually or prolong it for specific user. Each license can be used only on limited machine number. This is done in order to protect script from unlicensed copying. Premium users can change machines number individually for each license.
